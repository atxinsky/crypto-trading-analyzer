# 币安交易数据高级分析系统

## 系统简介

币安交易数据高级分析系统是一个专为加密货币交易者设计的数据分析工具，可以导入币安交易历史记录，进行数据清洗、整合和深度分析。系统提供专业级K线图表和交易标记功能，帮助交易者更好地了解自己的交易表现，识别优势和劣势，优化交易策略。

## 功能特点

1. **专业K线图表**
   - 使用Lightweight Charts (TradingView轻量版)实现高品质K线图表
   - 支持多种时间周期(日、4小时、1小时、30分钟等)
   - 买入/卖出点位直观标记，鼠标悬停可查看详情

2. **强化的订单面板**
   - 专业交易订单列表展示
   - 按账户/交易对汇总盈亏信息
   - 持仓时间分析与收益率计算

3. **综合交易统计**
   - 自动计算胜率、盈亏比、夏普比率等关键指标
   - 可视化盈亏分布与交易频率

4. **多维度数据分析**
   - 按交易对、时间、交易方向多角度分析
   - 盈亏热力图与趋势图表

## 安装与运行

### Windows用户

1. 双击运行`setup.bat`文件
2. 系统将自动安装所需依赖并启动应用
3. 如遇到权限问题，请右键点击`setup.bat`，选择"以管理员身份运行"

### Mac/Linux用户

1. 打开终端，导航到系统文件夹
2. 运行以下命令：
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```
3. 系统将自动安装所需依赖并启动应用

## 使用指南

1. **数据上传与处理**
   - 点击"上传币安CSV交易数据文件"，选择您从币安导出的CSV文件
   - 点击"处理上传的数据"按钮
   - 系统将自动合并、清洗数据并保存到桌面

2. **交易分析仪表盘**
   - 查看交易统计数据和图表
   - 了解交易分布和盈亏情况

3. **K线图表与交易标记**
   - 选择交易对和时间范围
   - 查看带有交易标记的K线图表
   - 悬停在交易标记上查看详情

4. **订单分析与统计**
   - 查看订单详情和统计数据
   - 分析交易表现

5. **盈亏分析**
   - 多维度分析盈亏情况
   - 查看最盈利和最亏损的交易

6. **交易策略评估**
   - 评估策略表现
   - 分析交易频率和时间分布

## 系统要求

- Python 3.7 或更高版本
- 500MB 以上可用磁盘空间（用于安装依赖）
- 最小屏幕分辨率：1280 x 720
